Concept

รับส่งข้อความด้วยเทคนิคการเขียน Ajax เพื่อลดการโหลดหน้าจอ เเละ เก็บข้อความไว้บนฐานข้อมูลตาราง Chat

DataBase

ER Diagram

ใช้ตาราง chat ตารางเดียว

1. id เก็บหมายเลขผู้ใช้เป็นค่าชนิด int ตั้งค่าเป็น AI ให้รันอัตโนมัติ เป็น primary key
2. Name ใช้เก็บชื่อคนเข้าใช้เป็นค่า varchar
3. email ใช้ email คนเข้าใช้ใช้เป็นค่า varchar
4. msg ใช้เก็บ massage ที่พูดคุยกันเป็น varchar
5. status เก็บข้อมูลออฟไลน์ int
6. Onchat เก็บสถานะออนไลน์ int
7. date วันที่เวลาที่ใช้งาน timestamp

How to do it



1. Step one Create DB

สร้างไฟล์ขึ้นมากตั้งชื่อว่า db.php ไฟล์นี้สร้างขึ้นมาเพื่อเชื่อมต่อฐานข้อมูล host ให้เป็น "localhost"



2.Step two Create Main Class



ผมต้องการให้เมื่อผู้ใช้พิมพ์ให้เริ่มกระบวนการทำงานคำสั่ง session_start()
สิ่งที่ผมต้องการ $_POST ส่งข้อมูลขึ้นไป คือ $name $email $comment
ให้ตัวเเปร $query เป็นการหยิบข้อมูลขึ้นมาจากตาราง chat ในฐานข้อมูล

เมื่อมีการกระทำที่ enter ในไฟล์ login.php $check ไว้เก็บค่า name,email,status,onChat ถ้า $check = 0
ใหัเพิ่มข้อมูลเข้าใส่ตาราง chat ในฐานข้อมูลถ้า $check = 1 ให้เเก้ไขเป็นข้อมูลล่าสุดที่ส่งเข้า
ไปในตาราง chat ในฐานข้อมูลถ้ามีการกระทำที่ sent ให้ส่งค่า $name , $email ,0,1,$comment
ไปที่ chat กรณีต่อมาถ้ามีการกระทำที่ logout ให้ส่งค่า(post) name , email ไปเเต่ให้ msg หรือ
ให้ส่งค่าเป็น null offline = 1 ให้ฐานข้อมูลจำไว้ header ให้เปลี่ยนหน้าจาก chatpage ไปที่ login.php

เขียน script function chat_ajax ไว้ส่งข้อความเปิดด้วยไฟล์ chat.php ให้เเสดงค่า ชื่อเเละข้อความ
เเละ function user_ajax ไว้ส่งข้อมูลผู้ใช้ที่ออนไลน์เปิดไฟล์ userOnline.php ไว้สำหรับเเสดงคนออนไลน์อยู่
