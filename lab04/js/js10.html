6: สร้างตัวแปร AR เป็น Array
โดยที่ ARช่องที่0 เก็บข้อความ ปี
โดยที่ ARช่องที่1 เก็บข้อความ วัน
โดยที่ ARช่องที่2 เก็บข้อความ เดือน
สร้างตัวแปร i = 0
สร้าง function next
ตัวแปร i เท่ากับ i เพิ่มค่าทีละ 1 หารเอาเศษกับความยาวของ Array ที่ชื่อ AR
นำข้อมูลใน Array มาใส่แทรกลงไปตามตำแหน่ง i ที่เพิ่มขึ้น โดย id = click (คลิก 1 ครั้ง เปลี่ยนข้อความ 1 ครั้ง โดยเริ่มคำจาก AR[1]ก่อน)
เมื่อคลิ้กที่คำว่า Default หรือบริเวณ div ID="click" โปรแกรมจะสั่งให้ function next ทำงาน
